# ShelfIQ

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [Docker](https://www.docker.com/) (optional, for containerized deployment)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/madhanmohan-sigmoid/ShelfIQ
   cd ShelfIQ
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   ```bash
   cp .env.example .env
   ```

4. **Run the Application**
   ```bash
   npm run dev
   ```

## Usage

Once the server is running, navigate to `http://localhost:3000` in your browser. Log in with your administrator credentials to begin setting up your product categories and warehouse locations.
